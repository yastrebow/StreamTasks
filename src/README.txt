Задача по стримам:
Создай класс Person c полями:
Integer id,
String name,
String lastName,
String patronymicName,
Integer age,
Nationality nationality.
Подумай, какие методы класса Object нужно переопределить у объекта, который переносит данные и переопредели их.
Nationality - перечисление с несколькими (3-4) известными тебе национальностями
Создай список с 10-15 разными людьми, но так, чтобы у некоторых из них совпадало имя, у некоторых совпадали инициалы
(Фамилия И.О.), у некоторых был одинаковый возраст, некоторые - одной национальности и несколько абсолютно одинаковых людей

Часть 1. Решение только циклами. Каждый пункт - отдельный метод.
1. вернуть список людей старше 18
2. вернуть средний возраст людей в списке
3. вернуть структуру вида "национальность - количество людей этой национальности". Национальности брать не из перечисления, а только из тех, которые есть у людей в списке
4. Вернуть список полных ФИО всех людей
5. Вернуть список всех инициалов (Фамилия И.О.)
6. вернуть структуру вида "фамилия - список людей с этой фамилией"
7. вернуть только уникальных людей, id не должен считаться.
8. вернуть список отсортированных по фамилии людей
9. Вернуть всех людей переданной в метод национальности
